Freshman year: Choose your own adventure: The Hob's Bit

In this project I was paired with another developer to create an end of the year project in C. Using the Windows graphic library we created a click based game that took the user on an adventure that presented them with choices that lead them down different paths based on their answers. It also had several minigames throughout the story to add skill challege to the players. My role is this project was lead developer and story board architect. My partner was the graphic designer and second developer. He made the scenes for the the story and created some minigames as well.
